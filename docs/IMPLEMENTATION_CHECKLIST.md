## ✅ QUESTION 2: MODEL SELECTION - HAIKU vs SONNET

### Use SONNET ✅

| Factor | Why Sonnet |
|--------|-----------|
| **Code quality** | Catches subtle CSS/JS issues |
| **Design polish** | Professional, production-ready |
| **Metrics accuracy** | Precise (500K RPS, not "around 500K") |
| **Navigation logic** | Handles complex branching correctly |
| **Component design** | Better reusable component structure |
| **Cost** | $6-9 vs $2-3 (worth it for multi-year use) |
| **Quality delta** | 70% (Haiku) vs 95%+ (Sonnet) |

**Decision:** SONNET for all HTML/CSS/JS generation

This guide will be used for **years of interview prep**. Extra $3-6 investment is worth the quality difference.
